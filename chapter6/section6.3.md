# 6.3. 非常直接教示

ロボットが活性化中に他の物体に触れていると、外力を受けたと考えて活性化になりません。 非常直接教示は、その状態を回避する機能です。



{% stepper %}
{% step %}
通信接続画面で右上の設定アイコンを押します。


{% endstep %}

{% step %}
「方法3」 非常直接教示を押します。


{% endstep %}

{% step %}
準備完了のメッセージが表示するまで待ってから、ロボットの直接教示ボタンを押したまま回避します。


{% endstep %}

{% step %}
回避完了後、ロボットアームの電源を落とします。


{% endstep %}

{% step %}
活性化を進めます。


{% endstep %}
{% endstepper %}
