# 8.1. 教示方法

教示ボタンを押して、ロボットの現在座標をプログラムの該当の場所に保存できます。

{% hint style="success" %}
**正しい教示方法**

チップの端から溶接ワイヤを15mmほど露出させた後、ワイヤの端点を溶接を希望する位置に合わせて教示を行ってください。
{% endhint %}
