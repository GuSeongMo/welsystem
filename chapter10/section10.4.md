# 10.4. プログラムの読み取り

#### ■ 直接読み取る方法

{% stepper %}
{% step %}
メイン画面から「プログラム選択」を押します。
{% endstep %}

{% step %}
「読み取り」ボタンを押せばウィンドウエクスプローラが表示されます。
{% endstep %}

{% step %}
DATAフォルダの中のプログラムを選択して開きます。
{% endstep %}

{% step %}
設定完了を選択して読み取ることが出来ます。
{% endstep %}

{% step %}
複数のプログラムを読み取ることも可能です。
{% endstep %}

{% step %}
この場合、上から順番に行っていきます。
{% endstep %}
{% endstepper %}

***

#### ■ 読み取りモードの使用

{% stepper %}
{% step %}
画面の左上の「読み取りモード」ボタンを選択します。
{% endstep %}

{% step %}
実行させるプログラムの番号を選択した後、設定完了を行います。
{% endstep %}
{% endstepper %}
