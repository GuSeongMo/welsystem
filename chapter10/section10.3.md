# 10.3. プログラムの保存

#### ■ 直接保存する方法

{% stepper %}
{% step %}
ファイル名欄を選択します。
{% endstep %}

{% step %}
名前を入力して、キーパッドを閉じます。
{% endstep %}

{% step %}
保存押します。
{% endstep %}

{% step %}
経路を指定し保存して完了します。
{% endstep %}
{% endstepper %}

***

#### ■ 保存モードの使用

{% stepper %}
{% step %}
左上の保存モードを押します。
{% endstep %}

{% step %}
保存する番号を選択してください。
{% endstep %}

{% step %}
確認メッセージからYESを選択します。
{% endstep %}

{% step %}
保存が完了します。「戻る」ボタンで前の画面に戻ります。
{% endstep %}
{% endstepper %}
